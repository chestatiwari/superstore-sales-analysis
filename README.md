# 📌 Project Overview

This project analyzes retail sales data from a US-based Superstore using **MySQL** for data exploration and **Microsoft Excel** for dashboard creation.

The objective is to uncover sales trends, profitability patterns, customer insights, and business opportunities through SQL queries and interactive visualizations.

---

# 🎯 Objectives

The analysis focuses on answering business questions such as:

- Which product category generates the highest revenue?
- Which products contribute the highest and lowest profit?
- How do monthly sales change over time?
- Which region performs best?
- How do discounts affect profitability?
- Which customer segment contributes the most revenue?
